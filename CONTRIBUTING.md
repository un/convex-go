# Contributing

Contributions are welcome.

Please share general questions, feature requests, or product feedback in the
[Convex Discord Community](https://convex.dev/community).

Before opening a PR:

- Format Go code with `gofmt`.
- Format markdown with `prettier`.
- Run tests with `go test ./...`.
- Run race checks with `go test -race ./...`.

Convex is a fast-moving project with a small team and broad surface area. For
larger changes, please discuss your approach in Discord first so we can align
on direction and avoid duplicated work.
